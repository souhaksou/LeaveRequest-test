Privacy & Security Policy

Last updated: 2026-01-01

Data Storage

All leave request data is stored exclusively in your browser's localStorage under the key leave-requests-v1. No data is transmitted to any server. Clearing browser storage will permanently remove all records.

Data We Collect

This application does not collect, transmit, or share any personal data. Everything runs client-side in your browser.

CSV Import / Export

CSV files are processed entirely in-browser. Exported values are hardened against formula injection attacks. Imported files are validated and limited to 2 MB.

PDF Export

PDF generation happens in-browser using jsPDF. The generated file is downloaded directly to your device without being uploaded anywhere.

Security

  • No external API calls are made.
  • Seeded demo data uses Faker and does not represent real individuals.
  • Exported CSVs are sanitised against formula-injection characters.
  • Static assets are served with long-cache headers; HTML pages require revalidation.

Contact

For questions, please open an issue on the project repository.
